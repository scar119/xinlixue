## 一、产品概述

### 1.1 产品简介

**产品名称**: 暂定 "心语AI" (HeartWords AI)

**一句话描述**:
用简单心理学与 AI 工具带来幸福，激发内在力量的互动平台

**产品定位**:
介于专业心理咨询和娱乐心理 APP 之间的"轻量专业"心理成长工具

**核心价值**:
- 简单：3 步上手，不需要心理学知识
- 有趣：游戏化互动，像玩游戏一样
- 有效：基于实证心理学，真实感受改变
- 陪伴：24/7 随时在身边

### 1.2 产品愿景

**短期愿景** (6 个月):
成为中文世界最受欢迎的心理学互动工具网站，月活用户 10,000+

**中期愿景** (2 年):
打造心理成长生态平台，覆盖 100 万+ 用户

**长期愿景** (5 年):
让每个人都能轻松获得心理支持，推动心理健康普惠化

### 1.3 目标市场

**主要市场**: 中国大陆
**次要市场**: 海外华语地区

**市场规模**:
- 潜在用户：2 亿（有心理提升需求的网民）
- TAM (Total Addressable Market): ¥200 亿/年
- SAM (Serviceable Addressable Market): ¥20 亿/年
- SOM (Serviceable Obtainable Market): ¥2 亿/年

### 3.2 MVP 核心功能 (Must Have)

#### 功能 1: 夸夸 AI ⭐⭐⭐⭐⭐

**功能描述**:
用户输入任何内容（想法、困扰、成就等），AI 给予真诚、具体的夸奖和正向反馈。

**用户流程**:
```
1. 用户在首页看到"来夸夸我"按钮
2. 点击进入夸夸页面
3. 输入框提示："今天想分享什么？"
4. 用户输入文字（10-500 字）
5. AI 分析并生成夸奖内容
6. 展示夸奖内容（动画效果）
7. 用户可以：分享、收藏、再来一次
```

**功能需求**:
| 需求ID | 需求描述 | 优先级 | 验收标准 |
|--------|---------|--------|----------|
| F-001 | 文本输入 | P0 | 支持 10-500 字输入 |
| F-002 | AI 情感分析 | P0 | 准确识别情感倾向 |
| F-003 | 夸奖生成 | P0 | 内容真诚、具体、不重复 |
| F-004 | 结果展示 | P0 | 动画效果，视觉美观 |
| F-005 | 分享功能 | P0 | 一键分享到社交平台 |
| F-006 | 收藏功能 | P0 | 保存到个人中心 |

**非功能需求**:
- 响应时间：< 3 秒
- 准确率：情感分析 > 85%
- 可用性：99.5%

#### 功能 2: 心理测试 ⭐⭐⭐⭐⭐

**功能描述**:
提供专业心理测试和趣味性格测试，AI 生成个性化报告。

**测试类型**:
1. **专业测试**
   - MBTI 性格测试
   - 抑郁自评量表 (PHQ-9)
   - 焦虑自评量表 (GAD-7)
   - 压力水平测试

2. **趣味测试**
   - 你是什么类型的"打工人"？
   - 你的隐藏超能力是什么？
   - 你的心理年龄是多少？

**用户流程**:
```
1. 测试列表页（分类展示）
2. 选择测试 → 测试介绍页
3. 开始测试（逐题作答）
4. 提交 → AI 分析中（加载动画）
5. 测试报告页（结果+解释+建议）
6. 分享/保存/其他测试
```

**功能需求**:
| 需求ID | 需求描述 | 优先级 | 验收标准 |
|--------|---------|--------|----------|
| F-007 | 测试列表 | P0 | 分类清晰，筛选方便 |
| F-008 | 测试引擎 | P0 | 支持单选、多选、量表 |
| F-009 | AI 报告生成 | P0 | 个性化、专业、易懂 |
| F-010 | 结果对比 | P1 | 与同龄人对比 |
| F-011 | 历史记录 | P1 | 查看历史测试结果 |

## 六、数据需求

### 6.1 数据实体

**用户**:
```
- id: UUID
- email: string (unique)
- password_hash: string
- nickname: string
- avatar: string?
- created_at: timestamp
- subscription_type: enum (free, premium)
```

**夸夸记录**:
```
- id: UUID
- user_id: UUID (nullable)
- input_text: text
- sentiment: enum (positive, neutral, negative)
- response_text: text
- created_at: timestamp
```

**测试**:
```
- id: UUID
- title: string
- description: text
- category: string
- questions: jsonb
- is_premium: boolean
```

**测试结果**:
```
- id: UUID
- user_id: UUID
- test_id: UUID
- answers: jsonb
- result: jsonb
- created_at: timestamp
```

### 6.2 数据流

```
用户输入 → AI 处理 → 结果展示
   ↓           ↓
记录日志    更新画像
```

### 6.3 数据保留策略

| 数据类型 | 保留期 | 删除策略 |
|---------|--------|----------|
| 用户数据 | 注销后 30 天 | 软删除 |
