# Cog.md - 产品认知框架

## 概述

本文档定义"心理学-AI工具站点"的核心认知框架，包括产品定位、核心概念、理论模型和解决方案。

---

## 一、核心定位

### 1.1 产品定位陈述

```
我们为追求心理成长的现代人提供
简单、有趣、有效的 AI 心理学工具
让他们在碎片化时间获得即时支持和正向反馈
从而激发内在力量，成为更好的自己
```

### 1.2 品牌核心价值

| 价值 | 说明 | 用户体验 |
|------|------|----------|
| **简单** | 不需要专业知识 | 3 步上手，秒开即用 |
| **有趣** | 游戏化互动 | 像玩游戏一样开心 |
| **有效** | 基于心理学实证 | 真实感受到变化 |
| **陪伴** | 24/7 随时在身边 | 不孤单的成长伙伴 |

### 1.3 差异化定位

**与心理咨询区别**:
- 不是治疗，是成长
- 不是被动接受，是主动探索
- 不是昂贵服务，是普惠工具

**与心理 APP 区别**:
- 不是内容平台，是互动工具
- 不是标准化产品，是个性化体验
- 不是单一功能，是成长生态

## 三、理论框架

### 3.1 核心心理学理论

**1. 积极心理学 (Positive Psychology)**

核心理念：关注人的优势和潜能，而非问题和缺陷

应用：
- 夸夸功能：发现和强化用户优点
- 成长记录：追踪积极变化
- 优势测试：识别个人优势

**2. 认知行为疗法 (CBT)**

核心理念：想法影响情绪和行为

应用：
- 思维重构工具
- 情绪日记
- 自动思维识别

**3. 自我决定论 (SDT)**

三大需求：
- **自主感** (Autonomy) - 自己做选择
- **胜任感** (Competence) - 感觉有能力
- **归属感** (Relatedness) - 与他人连接

应用：
- 个性化路径
- 渐进式挑战
- 社区功能

**4. 心流理论 (Flow)**

心流状态：完全沉浸、忘记时间

应用：
- 难度自适应
- 即时反馈
- 目标清晰

### 3.2 游戏化理论

**八角行为分析框架 (Octalysis)**:

| 驱动力 | 应用场景 |
|--------|----------|
| **史诗意义** - 使命感 | "成为更好的自己" |
| **进步成就感** - 成长 | 等级、徽章、里程碑 |
| **创意反馈** - 表达 | 日记、创作工具 |
| **所有权** - 拥有 | 个人成长档案 |
| **社交影响** - 连接 | 分享、社区 |
| **稀缺性** - 珍贵 | 限时活动 |
| **不可预测性** - 惊喜 | 随机奖励 |
| **避免损失** - 完结 | 连续签到 |

## 四、功能框架

### 4.1 五大核心功能模块

**1. 夸夸模块**
- AI 分析用户输入
- 生成真诚的夸奖
- 发现隐藏优点
- 提供具体证据

**2. 游戏模块**
- 轻量心理游戏
- 寓教于乐
- 即时反馈
- 社交分享

**3. 实验模块**
- 心理学小实验
- 自我探索
- 行为改变
- 结果可视化

**4. 测试模块**
- 专业心理测试
- 趣味性格测试

- 个性化建议

**5. 理论模块**
- 心理学知识库
- 图文并茂
- 案例丰富
- 可搜索

- ⭐ 用户系统（沉淀）

**V2.0 阶段** (12 个月):

## 五、用户旅程

### 5.1 典型用户路径

```
【发现】
SEO/社交媒体 → 看到"免费心理测试"
   ↓
【吸引】
有趣标题 + 美观界面 → 点击尝试
   ↓
【参与】
简单 3 步 → 完成测试 → 获得报告
   ↓
【惊喜】
AI 分析很准 → 有共鸣 → 分享朋友圈
   ↓
【留存】
收藏网站 → 第二天再来 → 尝试夸夸

【付费】

   ↓
【推荐】
很满意 → 推荐给朋友 → 病毒传播
```

| 首次 → 7 日留存 | 每日提醒 + 内容更新 | 40% |

## 六、AI 应用策略

### 6.1 AI 的三种角色

**1. 专家角色** (Knowledge)
- 提供心理学知识
- 解答用户疑问
- 推荐合适方法

**2. 伙伴角色** (Companion)
- 倾听和共情
- 给予正向反馈
- 陪伴成长

**3. 引导者角色** (Guide)
- 设计挑战任务
- 提供行动建议
- 追踪进展

### 6.2 AI 能力矩阵

| 能力 | 优先级 | 技术方案 |
|------|--------|----------|
| **情感理解** | 高 | Fine-tuned 情感分类 |
| **文本生成** | 高 | GPT-4 + Prompt 工程 |
| **个性化推荐** | 中 | 协同过滤 + 内容相似度 |
| **语音交互** | 低 | 语音识别 + TTS |
| **图像生成** | 低 | DALL-E / Midjourney API |

## 七、商业模式

### 7.1 三阶段变现模型

**第一阶段：流量变现** (0-6 个月)
- Google AdSense
- 联盟营销
- 目标：覆盖基础成本

**第二阶段：Freemium** (6-12 个月)
- 免费版：基础功能
- 会员版：高级功能 + 无广告
- 价格：¥9.9 - ¥29.9/月

**第三阶段：生态变现** (12 个月+)
- 企业培训服务
- 心理咨询师对接
- 数据洞察服务

### 7.2 单位经济模型

**获客成本 (CAC)**:
- SEO 流量：¥5/人
- 内容营销：¥10/人
- 平均：¥8/人

**生命周期价值 (LTV)**:
- 免费用户：¥20 (广告)
- 付费用户：¥200 (订阅)
- 付费率 5%
- 平均 LTV：¥29.5

**LTV/CAC**: 3.7x (健康)

## 九、品牌人格

### 9.1 品牌拟人化

如果品牌是一个人，他/她是：

- **年龄**: 28 岁（成熟但有活力）
- **性格**: 温暖、专业、有趣
- **语气**: 像朋友聊天，不是说教
- **价值观**: 相信每个人都有潜力
- **口头禅**: "你其实很棒"

### 9.2 设计语言

**色彩心理学**:
- 主色：暖橙色（温暖、活力）
- 辅色：柔和蓝（平静、信任）
- 点缀色：亮黄色（乐观、希望）

**设计原则**:
- 圆角设计（亲和力）
- 渐变背景（现代感）
- 卡片布局（清晰）
- 留白充足（舒适）

## 总结

**核心认知**:
> 不是做一个心理 APP，而是打造一个"AI 时代的心灵成长伙伴"
> 用技术让心理学变得简单、有趣、有效

**产品哲学**:
- **简单**: 复杂留给自己，简单留给用户
- **真诚**: AI 不是替代人类，是连接人性
- **成长**: 用户和产品一起成长

**实施原则**:
- 快速迭代
- 数据驱动
- 用户第一
- 保持敬畏（对心理学、对用户）
